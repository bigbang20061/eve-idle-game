# 游戏深度设计

## 核心循环

玩家设定作业类型、目标星系和风险阈值。服务端根据舰船、模块、技能和星系 SDE 数据生成异常点。成功撤离后，货物入库，并按保留量卖出超额库存。

## 深度来源

- 星系：安等越低，收益和风险越高。
- 舰船：货舱、矿仓、DPS、扫描、撤离速度、反跳稳定影响作业路线。
- 模块：从 SDE group/marketGroup/name 派生槽位和效果。
- 蓝图：从 SDE blueprints 派生材料和产物，形成工业循环。
- 市场：basePrice、星系安等、每日随机波动形成价差。
- 囤积：保留量决定哪些材料留给制造，哪些自动卖出。

## 中长期目标

- 高安：稳定采矿/运输，适合囤材料。
- 低安：遗迹/数据点收益上升，风险和撤离判断更重要。
- 零安/虫洞：舰队收益高，单人损船概率高。
- 工业玩家：囤积矿物和打捞件，利用蓝图制造舰船/装备。
- 市场玩家：根据星系价格波动买卖库存。
